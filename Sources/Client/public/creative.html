Creative Coding

Where Code Meets Art

Creative coding is about expressing ideas through computation. I build interactive visual experiences that blend physics accuracy with artistic intent — from particle simulations to animation engines that bring mathematical concepts to life.

Physics Simulations

Real-time physics engines require balancing accuracy with performance. I implement rigid body dynamics, collision detection and response, and constraint solvers that feel natural and responsive to user input.

  • Rigid body dynamics with impulse-based collision resolution
  • Spatial partitioning for efficient broad-phase detection
  • Verlet integration for stable, energy-conserving simulations
  • Interactive drag, throw, and manipulation of physics objects

Animation Systems

Inspired by libraries like Manim, I've built ECS-based animation frameworks that support complex scene composition, timeline scrubbing, and interactive mobject manipulation. These tools make it possible to create rich, programmatic animations for education and visualization.

The Creative Process

Every creative project starts with curiosity. What happens if particles follow sine waves? How does a spring system handle 1000 interconnected nodes? The answers always surprise me, and the journey from question to interactive demo is where the real learning happens.
